## 📋 About This Repository

This repository eliminates the need to start from scratch on every new project. It is the single source of truth for WPQ project execution — housing proven solutions, templates, and lessons learned from prior work across all service lines.

**What you'll find in each module:**

- Processes and procedures
- Equations and calculation templates
- Document templates (BOMs, design reports, specifications)
- Applicable codes and standards
- Certifications and regulatory requirements
- Software tools and references
- Required project documents
- Past project records and lessons learned
