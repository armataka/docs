## Processes and Procedures

### Maintenance Engineering Procedures

**1. Maintenance Strategy Development (RCM)**

- Define system boundaries and functional hierarchy
- Perform Failure Mode and Effects Analysis (FMEA) for critical equipment
- Apply RCM decision logic to determine maintenance task type (preventive, predictive, corrective, redesign)
- Develop maintenance task list with frequencies and skill requirements
- Establish PM programme in CMMS
- Review and optimise strategy based on failure data and operational experience

**2. Planned Preventive Maintenance (PM) Procedure**

- Generate PM work orders from CMMS based on time/running-hours triggers
- Issue work order with scope, procedures, parts, tools, and permits
- Execute PM task per standard maintenance procedure (SMP)
- Record findings, measurements, and as-found conditions
- Close work order with as-left condition and recommendations
- Feedback to maintenance strategy for frequency optimisation

**3. Predictive Maintenance / Condition Monitoring Procedure**

- Define condition monitoring techniques per equipment type:
  - Vibration analysis (rotating machinery)
  - Oil analysis (gearboxes, hydraulics, engines)
  - Thermography (electrical, mechanical, process equipment)
  - Ultrasonic thickness measurement (piping, vessels, structures)
  - Performance monitoring (compressors, turbines, pumps)
- Establish baseline readings and alarm/trip limits
- Route-based data collection at defined frequencies
- Trend analysis and P-F interval management
- Generate corrective work orders when thresholds are approached

**4. Corrective / Breakdown Maintenance Procedure**

- Receive and log failure notification
- Prioritise based on safety, environmental, production, and cost impact
- Diagnose root cause
- Plan and execute repair (materials, permits, resources)
- Post-repair testing and return to service
- Root Cause Analysis (RCA) for significant failures (5 Whys, Fishbone, Fault Tree)
- Update failure history in CMMS

**5. Spare Parts Management Procedure**

- Develop spare parts list based on RCM output and OEM recommendations
- Classify spares: insurance/critical, operational, consumable
- Define min/max stock levels and reorder points
- Manage warehouse with identification, preservation, and shelf-life tracking
- Obsolescence management and alternative sourcing
- Annual spare parts review and cost optimisation

**6. Asset Integrity Management Procedure**

- Define asset integrity management framework (barriers, performance standards)
- Establish inspection regime per asset type (per API 580/581 RBI or time-based)
- Monitor Safety Critical Elements (SCEs) against performance standards
- Track integrity anomalies, deferrals, and overdue inspections
- Report asset integrity KPIs to management
- Annual review and audit of AIM programme

### Maintenance Execution Procedures

**7. Bolt Tensioning and Torqueing Procedure**

- Identify bolted joint criticality classification
- Select tensioning method: hydraulic bolt tensioner or torque wrench
- Define target bolt load / torque value per flange class, bolt grade, and gasket type
- Execute tensioning in approved cross-pattern sequence
- Record applied tension/torque for each bolt position
- Leak test after assembly where required

**8. Corrosion Monitoring and Mapping Procedure**

- Define corrosion monitoring locations (based on RBI and process conditions)
- Techniques: UT thickness measurement, corrosion coupons, ER probes, LPR probes
- Data collection frequency based on corrosion rate and remaining life
- Corrosion rate calculation and remaining life assessment
- Update corrosion maps with new survey data
- Generate integrity alerts when approaching minimum allowable thickness
- Feed data into RBI reassessment

**9. Blasting, Painting, and Coating Procedure**

- Surface preparation to specified standard (Sa 2.5 per ISO 8501-1)
- Environmental monitoring before and during application
- Apply coating system per approved specification (primer, intermediate, topcoat)
- Measure WFT/DFT at each coat
- Holiday detection for immersed/buried coatings
- Cure time verification before handling or service
- Inspection and documentation per NORSOK M-501

**10. Offshore Crane Maintenance Procedure**

- Planned maintenance per OEM schedule and DNV/API 2C requirements
- Annual load test and NDT of structural components
- Wire rope inspection and replacement criteria
- Slew ring, boom, and hook block inspection
- Hydraulic system maintenance (oil analysis, filter changes, leak checks)
- Crane operator daily pre-use checks
- Annual third-party certification inspection

## Equations and Calculation Templates

| Calculation | Method | Application |
| --- | --- | --- |
| MTBF | MTBF = Total operating time / Number of failures | Equipment reliability |
| MTTR | MTTR = Total repair time / Number of repairs | Maintenance efficiency |
| Availability | A = MTBF / (MTBF \+ MTTR) x 100% | Asset performance |
| Corrosion rate (short term) | CR = (T\_previous - T\_current) / time interval (mm/yr) | Integrity assessment |
| Remaining life | RL = (T\_current - T\_min) / Corrosion rate | Fitness for service |
| P-F interval | Time between detectable failure onset (P) and functional failure (F) | CBM frequency setting |
| Bolt torque | T = K x d x F where K = nut factor, d = bolt diameter, F = target clamp load | Bolted joint integrity |
| Hydraulic tensioner load | F = P x A where P = hydraulic pressure, A = piston area | Bolt tensioning |
| Spare parts reorder point | ROP = (Average daily usage x Lead time) \+ Safety stock | Inventory management |
| RBI probability of failure | Based on damage mechanism assessment, inspection effectiveness, and time in service | API 581 |
