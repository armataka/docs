## Processes and Procedures

**1. NDT Procedure Development and Execution**

- Select NDT method based on defect type, material, geometry, and access:
  - Visual Testing (VT)
  - Magnetic Particle Testing (MPI/MT)
  - Ultrasonic Testing (UT) — conventional and phased array (PAUT)
  - Radiographic Testing (RT) — film and digital (CR/DR)
  - Liquid Penetrant Testing (PT/DPI)
  - Eddy Current Testing (ECT)
  - Time-of-Flight Diffraction (TOFD)
  - Alternating Current Field Measurement (ACFM)
- Develop NDT procedure per ISO 9712 / ASNT SNT-TC-1A requirements
- Procedure qualification: demonstrate on reference standard or mock-up
- Equipment calibration and verification
- Surface preparation requirements
- Acceptance criteria per applicable code (ASME V, API 1104, AWS D1.1, BS EN ISO 5817)
- Reporting format and archival

**2. Welding Inspection Procedure**

- Review WPS/PQR and welder qualification records (WQR)
- Pre-weld inspection: fit-up, alignment, preheat, material traceability
- In-process monitoring: interpass temperature, electrode/wire control, visual inspection of each pass
- Post-weld inspection: visual examination, dimensional check, NDT per code requirements
- Post-weld heat treatment (PWHT) verification where required
- Documentation: weld map, weld log, NDT reports, repair logs
- Acceptance per project code: ASME IX, API 1104, AWS D1.1, BS EN ISO 15614

**3. Coating Inspection Procedure**

- Surface preparation inspection: cleanliness (SA 2.5 / SSPC-SP10), profile (Testex tape), dust, salts (Bresle method)
- Ambient conditions monitoring: temperature, RH, dew point, steel temperature
- Wet film thickness (WFT) measurement during application
- Dry film thickness (DFT) measurement per coat and total system
- Holiday/pinhole detection (low voltage wet sponge or high voltage spark test)
- Adhesion testing (cross-cut or pull-off per ISO 4624)
- Cure/hardness testing where specified
- Inspection per NORSOK M-501, SSPC-PA2, ISO 8501/8502/8503

**4. Pressure Equipment Inspection Procedure**

- Establish inspection scope from equipment register and RBI study
- External visual inspection: corrosion, distortion, support condition, insulation condition
- Internal inspection (where possible): corrosion mapping, UT thickness survey
- Thickness measurement locations per API 510/570 CML (Condition Monitoring Location) plan
- Remaining life calculation: corrosion rate x (current thickness — retirement thickness) / corrosion rate
- Fitness-for-service assessment per API 579-1 / ASME FFS-1 where minimum thickness is breached
- Re-rating calculations where operating conditions have changed
- Statutory compliance: pressure test or alternative (risk-based) where required
- Issue inspection report with findings, recommendations, and next inspection date

**5. Lifting Equipment Inspection Procedure**

- Inventory all lifting equipment and accessories: cranes, davits, slings, shackles, pad eyes, spreader bars
- Thorough examination per LOLER 1998 intervals (6 months for accessories, 12 months for equipment)
- Load testing where required (proof load = 1.25x SWL for cranes, 2x SWL for slings/shackles)
- Colour-coding system for inspection status (quarterly colour rotation)
- Defective equipment quarantine and disposal procedure
- Crane structural inspection: boom, pedestal, slew ring, wire ropes
- Certification and register update

**6. Pipeline Inspection Procedure**

- In-line inspection (ILI/intelligent pigging): MFL, UT, caliper, geometry
- Pre-ILI: cleaning runs, gauge plate runs
- ILI data analysis: metal loss features, dents, laminations, girth weld anomalies
- Direct assessment methods: ECDA, ICDA, DCVG/CIPS for cathodic protection
- Above-ground inspection: pipe supports, expansion loops, coating condition, flanges, valves
- Fitness-for-service: ASME B31G / RSTRENG / DNV-RP-F101 for corrosion features
- Repair/replacement decision and prioritisation
- Reporting per API 1163 (ILI system qualification)

**7. Subsea Inspection Procedure (ROV-based)**

- General visual inspection (GVI) of subsea structures, pipelines, risers
- Close visual inspection (CVI) with HD camera at identified anomalies
- Cathodic protection survey (anode wastage measurement, potential readings)
- Marine growth assessment and cleaning requirements
- Flooded member detection (FMD) for jacket structures
- Scour and span surveys for pipelines
- Reporting per DNV-RP-F116 (integrity management of subsea pipeline systems)

**8. Third-Party Inspection and Expediting**

- Vendor/manufacturer surveillance during fabrication
- Stage inspections per ITP (Inspection and Test Plan)
- Material verification: review MTRs, PMI testing
- Dimensional inspection and fit-up verification
- Witness testing: hydrostatic, pneumatic, FAT (Factory Acceptance Test)
- Expediting: monitor vendor schedule, flag delays, issue progress reports
- Final inspection, packing, and release for shipment

## Equations and Calculation Templates

| Calculation | Method | Application |
| --- | --- | --- |
| Remaining life (pressure equipment) | RL = (t\_actual — t\_min) / corrosion rate | API 510/570 inspection planning |
| Maximum allowable working pressure (MAWP) | MAWP = (S x E x t) / (R \+ 0.6t) per ASME VIII Div 1 | Pressure vessel re-rating |
| Pipeline burst pressure (ASME B31G) | Pb = 2St/D x (1 — A/Ao) / (1 — A/(Ao x M)) | Pipeline corrosion assessment |
| Minimum required thickness | t\_min = PR / (SE — 0.6P) for cylindrical shells | Equipment fitness-for-service |
| Corrosion rate (short-term) | CR = (t\_previous — t\_current) / time interval | Thickness monitoring |
| Corrosion rate (long-term) | CR = (t\_nominal — t\_current) / service years | When no prior readings exist |
| Remaining strength factor (RSF) | RSF = actual strength / original strength per API 579 | Locally thinned area assessment |
| Holiday detection voltage | V = 5 x DFT (volts) for coatings \>500 microns (high voltage) | Coating inspection |
