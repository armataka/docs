## 📄 Processes and Procedures

**1. Installation Engineering and Planning Procedure**

- Review design basis and installation analysis reports
- Develop installation procedures and sequences
- Identify required vessels, equipment, and spread
- Perform installation engineering: lift analysis, loadout analysis, seafastening design, upending analysis
- Weather window analysis for critical marine operations
- SIMOPS (Simultaneous Operations) assessment
- Develop detailed installation schedule (integrated with overall project schedule)
- Marine warranty surveyor (MWS) submission and approval

**2. Heavy Lift Procedure**

- Lift study: weight report, CoG determination, rigging arrangement, crane capacity verification
- Rigging plan: sling selection, shackle sizing, spreader bar design
- Lift factor of safety per API RP 2A / client specification (typically 2.0 dynamic, 1.5 static)
- Ground conditions assessment for onshore lifts (crane pad design)
- Tandem lift procedure where applicable
- Pre-lift meeting: roles, responsibilities, communication protocol, abort criteria
- Execution: trial lift, load monitoring, tagline management
- Post-lift verification: alignment, bolt-up, weld-out

**3. Jacket and Platform Installation Procedure**

- Loadout: skid, roll-on, or lift onto transportation barge
- Seafastening design and installation
- Marine transportation: tow route, weather routing, motion monitoring
- Launch or lift installation at site
- Pile driving: hammer selection, pile monitoring (blow count, set per blow), pile refusal criteria
- Grouting: annulus grout design, placement procedure, volume monitoring
- Topside lift and set: crane vessel positioning, alignment, set-down, securing
- As-installed survey

**4. Pipeline Installation Procedure**

- Method selection: S-lay, J-lay, reel-lay, tow, or pull
- Pipe preparation: coating, anode installation, field joint coating
- Lay vessel setup: tensioner, stinger, alignment, welding stations
- Pipeline welding (mechanised or manual) and NDT per procedure
- Lay rate monitoring and pipeline catenary/sag bend control
- Span rectification post-lay
- Tie-in operations: hot tap, flange, or weld
- As-laid survey: ROV inspection, cathodic protection survey
- Pre-commissioning: flooding, gauging, cleaning, hydrostatic testing

**5. Subsea Equipment Installation Procedure**

- Equipment preparation: factory acceptance test, system integration test
- Deployment method: vessel crane, through-moonpool, or A-frame
- Guidance system: acoustic, wire-guided, or ROV-guided
- Landing and leveling on pre-installed foundation or seabed
- Connection: collet connector, clamp connector, or hydraulic make-up
- Function testing: hydraulic, electrical, fiber-optic circuits
- As-installed survey and protection installation (rock dump, mattress)
- Handover to operations

**6. FPSO Hook-Up and Integration Procedure**

- Mooring system installation: anchors, chains, polyester lines, buoys
- FPSO positioning and mooring hook-up
- Turret or spread mooring connection
- Riser pull-in and connection
- Topside module integration: mechanical, electrical, instrument tie-ins
- Pre-commissioning and commissioning by system
- Progressive handover: system-by-system from construction to commissioning to operations

**7. Onshore Equipment Installation Procedure**

- Foundation preparation and verification (survey, hold points)
- Equipment receiving inspection
- Setting and alignment: optical survey, laser alignment
- Grouting: non-shrink grout, cure, and check
- Mechanical installation: piping, valves, instrumentation
- Electrical and instrument terminations
- Pre-commissioning checks per discipline
- Punch list generation and close-out

## 📝 Equations and Calculation Templates

| Calculation | Method | Application |
| --- | --- | --- |
| Crane lift capacity | Load chart deration for vessel motion (dynamic amplification factor = 1.1 to 2.0 depending on sea state and period) | Heavy lift planning |
| Sling tension | T = W / (n x cos θ), where W = load weight, n = number of slings, θ = angle from vertical | Rigging design |
| Pile driving: wave equation analysis | GRLWEAP or equivalent — hammer energy, cushion, soil resistance parameters | Pile installation |
| Pipeline on-bottom stability | Lateral stability: (Ws — FL) x μ ≥ FD + FI per DNV-RP-F109 | Pipeline design verification |
| Pipeline free span | Allowable span = f(OD, wall thickness, contents, current velocity, VIV) per DNV-RP-F105 | Span assessment |
| Seafastening design | Inertia loads from vessel motion (roll, pitch, heave accelerations per transport analysis) | Marine transportation |
| Ballast transfer | Required ballast = displaced weight x moment arm / tank moment arm | Barge stability during loadout |
| Cathodic protection design | Anode mass = (Current demand x design life) / (anode utilisation x electrochemical capacity) | CP system installation |

## ✅ Certifications and Regulatory Requirements

- API RP 2A platform installation compliance
- DPR/NUPRC installation permit and safety case approval

- DNV / ABS / Lloyds / BV class certification for vessels and barges
- Crane operator certifications (LEEA, CPCS, or equivalent)
- Rigging and slinging certifications
- BOSIET / HUET / FOET for offshore personnel
- Diving certification (IMCA / ADCI) for subsea installation support
- ROV pilot certifications (IMCA)
- DP operator certificates (Nautical Institute)
- NCDMB/NOGIC Act local content compliance
- NIMASA cabotage compliance for vessels
